* Introduction to git and github.
* Git :- it is an open source distributed version control system(VCS) or source code management(SCM).
       --> features of git
           * open source 
           * scalable
           * distributed
           * secure
           * speed
* Github :- Remote repository hosting service.
* Branch :- Creating a copy of the master branch or main branch to make our own changes.
* Mergeing :- Adding the modified or updated code of the user branch with the main branch.
* Basic commands of Git
  -git init -> to make a local git repository.
  -git status -> to check the status of the git repository.
  -git log  -> to get log about the git repository.
  -git push  -> to upload the files from local to remote repository.
